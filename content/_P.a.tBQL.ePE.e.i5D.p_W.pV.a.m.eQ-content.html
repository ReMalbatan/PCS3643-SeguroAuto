UseCase Vender seguro

Caso de uso: Vender seguro

Descrição: Este caso de uso descreve o processo de venda de um seguro

Evento iniciador: Usuário seleciona a opção orçar um seguro

Atores: Corretor, Comprador

Pré-condição: Sistema exibindo opções de interação com usuário

Seqüência de eventos:

1 Sistema apresenta formulário para preenchimento dos dados referente ao segurado

2 Sistema apresenta formulário para preenchimento dos dados referente ao veículo

3 Sistema efetua o cálculo do valor do prêmio

4 Sistema mostra para o usuário o orçamento da apólice gerada e oferece a opção de compra e cancelamento

5 O usuário seleciona a opção de comprar a apólice

6 O corretor insere os dados dele e da corretora na apólice

7 O cliente realiza o pagamento

8 A apólice é registrada no sistema com o status de ativo

Pós-condição: Apólice recém-vendida está registrada no sistema com o status de ativo

Extensões:

1 Usuário resolve não comprar aquele seguro: sistema retorna para a tela inicial(passo 4).

2 Usuário coloca veículo que não consta no banco de dados: sistema retorna uma mensagem de erro (passo 4)

3 Cliente não realiza o pagamento da apólice: A apólice não é registrada no sistema(passo 8)

Inclusões:

1 Caso de Uso registrar status da apólice (passo 8).
